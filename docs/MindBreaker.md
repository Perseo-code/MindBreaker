# The use of mindbreaker
MindBreaker is a very simple and autonomous shell script for linux. \
## What does it do?
The mindbreaker.sh file (The main file) is just a module loader. It just looks at the files inside the modules folder \
and gives you an option so you can run that module. You can also load them using bash. The whole point is to give many scripts \
that could help many beginners. For example, take a look at the msfvenom module, you don't need to know anything else than the location \ 
of the payloads. You don't even need to write a msfvenom command. Since it's very easy to use, I highly recommend new users to try it out. \

# How can I load modules?
If you don't know (which is weird), you just put the number of the script you want to use and then hit enter.
