# MindBreaker
MindBreaker is a simple, easy to use loader.

# Installation
To install this program, you have to follow very easy steps.

```
curl https://raw.githubusercontent.com/Perseo-code/MindBreaker/main/install.sh --output install
chmod +x ./install
sudo ./install
```

# How it works
Basically, it gives you a couple scripts to hack, and you can load them using the file mindbreaker, \
Or simply use bash 'module'. Basically, it brings some scripts for beginners. \
The scripts are normally a look-alike of <a href="https://github.com/rapid7/metasploit-framework">metasploit framework</a>

# How do I create scripts?
MindBreaker only executes bash scripts with the bash command internally. \
So just learn how to use bash and you got it. This repository is actually just scripts.
